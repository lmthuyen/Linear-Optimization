# Linear Optimization Model
## LP Solver - Browser-Based Linear Programming Tool
A browser-based linear programming solver built with vanilla HTML, CSS, and JavaScript. Developed as part of a Decision Models course project (BANA 4095), this app lets users set up and solve LP optimization problems entirely in the browser, no installation or backend required

**Features:**
* Supports any number of decision variables and constraints
* Toggle between maximization and minimization objectives
* Dynamically generated input fields based on problem size
* Displays optimal objective value and decision variable values
* Identifies binding vs. non-binding constraints
* Shows shadow prices and reduced costs for sensitivity analysis
* Handles infeasible and unbounded problems with clear error messages

**Tech Stack:**
* Vanilla HTML / CSS / JavaScript
* javascript-lp-solver via CDN (Simplex method)

**Usage:**
* Just open solver.html in any modern browser (Chrome, Firefox, Edge). No setup needed.

**Validated Against:**
* Product mix maximization
* Maximization with demand constraints
* Blending minimization
